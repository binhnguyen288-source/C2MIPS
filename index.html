Simple C to MIPS Compiler (WIP)

Welcome to a simple C to MIPS Compiler for Computer Architecture, since the compiler ain't completed yet so we only have a few options to choose from, the current only supported features are
  • Word integer arithmetic (maximum 10 register).
  • Word integer array (maximum 8).
  • Printing Immediate string.
Note that this is not even a full C compiler (yet). Only the assignment operator = is accepted, no pointer arithmetic has been implemented other than array member accessing, no floating number support, no function declaration, and the list go on. Also the compiler might be buggy sometimes so use it at your own risk. The program prewritten below show nearly all implemented features so far.
